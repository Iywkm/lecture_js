9. 繰り返し処理

・for文の基本形

for文は指定した回数だけ繰り返し処理を実施する.
記述方法は下記.
for (初期化式; 条件式; 変化式){
   実行する処理1;
   実行する処理2;
}
例.
for (let i = 0; i < 3; i++){
   console.log("i=" + i)
}
console.log("end")

>> i=0
>> i=1
>> i=2
>> end

また、if文と同様に繰り返し実行する処理が一つの場合は{}を省略できる.
for (初期化式; 条件式; 変化式)
   実行する処理;

・while文

while文は条件式がtrueの間だけ処理を実行する.
while (条件式){
   実行する処理1
   実行する処理2
}
例.
let x = 0
while (x < 3){
   console.log("x =" + x)
   x++
}
console.log("end")

>> x=0
>> x=1
>> x=2
>> end

for文と同様に処理が1文の場合は{}を省略可能.

・for...of文

for...of文は配列の要素を順に取得することができる.
記述方法は下記.
for (変数名 of 配列){
   実行する処理1;
   実行する処理2;
}
例.
const array = [1, 2, 3]
for (const el of array){
   console.log("element=" + el);
}

>> element=1
>> element=2
>> element=3

・for...in文

for...in文はオブジェクトのプロパティを順に取得することができる.
記述方法は下記.
for (変数名 in オブジェクト){
   実行する処理1;
   実行する処理2;
}
例.
const object = {"jan": 1,"feb": 2 ,"mar": 3}
for (const key in object){
   console.log("object." + key + "=" + object[key]);
}

>> object.jan=1
>> object.feb=2
>> object.mar=3
