3. 関数

定義方法

・functionキーワードでの定義

関数を定義する方法の一つとして、functionキーワードを使った宣言がある.
function 関数名(引数1, 引数2, ...) {
   処理
   return 戻り値
}
戻り値がない場合はreturnは省略可能(undefinedがreturnされる).

例.
function addTen(number) {
   console.log("The argument is", number);
   return number + 10;
}

function greeting() {
   console.log("hello");
}

・アロー関数(無名関数)

functionキーワードを使わない宣言方法.
() => {
   処理
   return 戻り値
}

処理がない関数の場合は{}を省略して記述できる.
() => 戻り値

そのまま定義する場合は名前のない関数として定義されるため、名前をつけたい場合は変数に格納する必要がある.
例.
const greeting = () => {
   console.log("hello");
}

関数の呼び出し

引数が必要な関数は、下記のように呼び出す.
関数名（引数, ...)
例.
addTen(5);

引数がない場合は下記のようになる.
関数名（）
例.
greeting();

関数からの戻り値を変数に格納する場合
const answer = addTen(5);
console.log(answer);
>> 15

関数の戻り値を関数の引数として渡すことも可能.
console.log(addTen(5));
>> 15
