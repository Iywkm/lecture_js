8. 条件分岐

if文を使って条件を評価した結果に応じて、異なる処理を実行できる.
記述方法は下記.
if (条件式) {
   条件式がtrueの場合の処理
}
この場合は、条件式がtrueの場合はブロック内の処理が実行され、falseの場合はブロック内はスキップされ次の処理に移る.

条件式がfalseの時の処理(if..else)

if (条件式) {
   条件式がtrueの場合の処理
} else {
   条件式がfalseの時の処理
}
この場合は条件式がtrueの時はifの直後のブロックの処理が、falseの場合はelseの直後の処理がそれぞれ実行される.

ブロックの{}を省略する記述

実行する処理が一つの文だけだった場合は{}を省略することができる.
if (条件式)
   条件式がtrueの場合の処理
else
   条件式がfalseの時の処理
