4. 配列

配列は複数の変数を一つの変数にインデックス（順番）と紐づけて保管できる

定義方法

[]で囲うことで記述できる.
const instructors = ["Kimiko", "Yan", "Dustin", "Dylan"];
const numbers = [1, 2, 3, 5, 7];

要素の取り出し

配列の要素を取り出すにはインデックスを指定する.
インデックスは0から始まる.
console.log(numbers[2]);
>> 3
console.log(numbers[0]);
>> 1

プロパティ

配列の要素数は.lengthでアクセスできる.これをプロパティと呼ぶ.
console.log(numbers.length - 1, "5");
console.log(numbers[numbers.length - 1], "7");

メソッド

配列にはあらかじめメソッドが用意されている.
メソッドを使いこなすことで、配列の要素を取り除いたり、追加したりする操作が楽に行える.
