5. オブジェクト

オブジェクトではデータをプロパティ名(キー)と値のペアで管理します.
下記のように、{}で囲うことで記述します.

const user = {
name: "yuki",
age: 30
}

オブジェクトではプロパティ名を指定することで値を取り出すことができます.
プロパティ名を指定する方法は下記のブラケット記法とドット記法があります.

console.log(user["name"])
>> yuki
console.log(user.age)
>> 30

オブジェクトには配列や関数を格納することもできます。

const user = {
name: "yuki",
age: 30
hobby: ["釣り", "キャンプ"]
getAge: function(){
    return this.age
  }
}

console.log(user.hobby[0])
>> 釣り
console.log(user.getAge())
>> 30
