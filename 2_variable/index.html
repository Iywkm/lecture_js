2. 変数

定義方法

・const

constは変更不可な変数を定義できる.
例.
const name = "yuki";
上書きしようとするとエラーが出る.
例.
name = "test";
TypeError: Assignment to constant variable.

・let

letは変更可能な変数を定義できる.
例.
let age = 30;
age = 31;
age = 32;

・var

変数の古い定義方法.
letよりも緩い宣言であり、スコープの概念がないため極力使わないほうが良い.
(スコープに関しては後述)

四則演算

四則演算の演算子は[+, -, *,/]を使う.
基本的にはpython等と違いはない.
