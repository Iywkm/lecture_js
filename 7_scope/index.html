7. スコープ

スコープとは変数や関数などの参照できる範囲を決めるもの.
スコープの中で定義されたものはスコープの内側からのみ参照でき、スコープの外側からは参照できない.

・関数スコープ

関数のブロック({と}の間)内で変数を定義する. この場合関数の中で変数を参照することはできるが、関数の外からの参照はエラーがでる.
function fn() {
   const x = 1;
   console.log(x);
}
fn(); => 1がコンソールに出力される.

console.log(x); => ReferenceError: x is not defined
このような関数によるスコープを関数スコープという.

・ブロックスコープ

{と}で囲んだ範囲をブロックと呼び、ブロックもスコープを作成する.
ブロック内で宣言された変数も同様に、スコープ内でのみ参照することができる.
if (true) {
   const x = 1;
   console.log(x); => 1がコンソールに出力される.
}

console.log(x); => ReferenceError: x is not defined
このようなブロックによるスコープをブロックスコープという.

・グローバルスコープ

グローバルスコープは名前の通り最も外側にあるスコープであり、プログラム実行時に暗黙的に作成される.
const x = "1";
console.log(x); => 1がコンソールに出力される.
グローバルスコープ内で定義された変数はグローバル変数であり、あらゆるスコープから参照可能となる.
